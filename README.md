## How to play
1. Go to [builds](builds) (NOTE: Only exists in dev branches so far)
2. Get the version you want and download the zip file
3. * For Windows: unzip the zip file and open the .exe file to run the game
   * For Linux: Type `unzip *filename*` to unzip, `chmod +x *filename*.x86_64` to make it executable and `./*filename*.x86_64` to run the game
  
## How to check out the project files
As this project is unobfuscated you can go to [msorberg-korlot-project/Assets](msorberg-korlot-project/Assets) and check out the assets. But most of the files can only be seen in the Unity Engine itself.
### How to check out the project in Unity
(This is done from a windows user's perspective)
1. Clone this repo
2. Install and open [Unity Hub](https://unity.com/download)
3. Go to [Unity download archive](https://unity.com/releases/editor/archive) and install `Unity 6000.4.0f1` to the hub
4. Select only the modules that you need and install (you don't need VS Community, and only platform build support if you want to share a new build)
5. Go to Unity hub and click `Projects` and click `Add` and then `Add project from disk` and go to the repo and add [msorberg-korlot-project](msorberg-korlot-project)
6. Open the newly initialised repo
7. DONE!
